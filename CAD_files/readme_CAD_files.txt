This folder contains the CAD parts necessary to reproduce the device.The 4 folders are:- F3D_files: contains the original CAD files in the Autodesk Fusion 360 format- STEP_files: contains the same CAD files but in the .stp format which can be open with any CAD software- DXF_laser_cutting: contains the vectorised 2D files (.dxf format) used to manufacture the layers on the laser cutting machine- STL_3D_printing: contains the CAD pieces that were 3D printed on a Prusa MK3S printer in the .stl format.
